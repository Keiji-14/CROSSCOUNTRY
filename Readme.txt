【制作使用エンジン】Unity

【制作期間】１ヶ月

【バージョン管理】SourceTree

【ゲームジャンル】３Ｄアクションゲーム

【ゲーム概要】
・このゲームは障害を避けながら跳ねて移動するエンドレスゲームです。

・プレイヤーをタップと各方向にフリックさせて移動させます。

・ステージには、「草原」「土」「コンクリート」「道路」「川」の５種類がランダムで出現します。

・「草原」と「土」と「コンクリート」のステージには（木、岩、建物）といった障害物があり、通れる場所を見つけて進みましょう。

・「道路」のステージには車やトラックが走っており、車と車の隙間をタイミングを見て通り抜けましょう。

・「川」のステージにはそのまま進むと溺れてしまうので、川に流れている丸太に乗り移って進みましょう。

・また、丸太に乗り続けると川に流されてしまうので急いで進もう。

・ステージ上に出現するコインを集めることによってガチャを回すことが出来ます。

・ガチャはスキン変更画面から選択することができ、コインが１０枚あればガチャを回す事ができます。

・ガチャを回す事で新たなスキンを手に入れることができ、プレイヤーのスキンが変更できるようになります。

・変更したスキンの種類によってはゲームステージがデフォルトの「草原ステージ」から「雪ステージ」、「砂漠ステージ」に切り替える事が出来ます。

【操作説明】
・前移動：画面タップ、前フリック
・右移動：画面右フリック
・左移動：画面左フリック
・後移動：画面下フリック

【アピールポイント・苦労した点】
・プレイヤーから前後左右にレイを出して、次に進む方向にステージの障害物があったら進めないように実装しました。

・オプション画面にある言語切り替え機能を実装し、他国の人にも遊べるように制作しました。

・空の切り替えでゲームの明るさを暗くして、車や建物に明かりを点けてゲームの雰囲気を良くなるように実装しました。

・スキンをScriptableObjectで生成したデータベースを利用して管理できるように実装しました。

・ScriptableObjectで「ガチャで出現するスキン情報のデータベース」と「スキン所持情報を保持するデータベース」を生成して簡易的に管理できるように実装しました。

・スキンを入手した後に、「スキン所持情報を保持するデータベース」に新しくスキン状態を保存して、スキン変更画面やスキンを変更した際にゲームなどに反映させる機能実装に苦労しました。

・変更したスキンに合ったステージを用意し、ゲームの見た目や雰囲気にこだわって制作しました。
